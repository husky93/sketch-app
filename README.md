# Pixel Sketch
Web app in which you can create pixelated art.

#### Features:
- Change brush color
- Switch canvas pixel size
- Change background color
- Pick color from canvas
- Eraser
- Clear canvas button
- Toggle pixel grid


Live: [click here](https://husky93.github.io/sketch-app/)
